[Eras ↗ ](https://eras.jaczerob.dev) is a web application that tracks information in real time and provides time-series
statistics for the game Toontown Rewritten. This application contains an Angular frontend hosted on Vercel and a Spring
Boot backend hosted on Fly.io.

Technologies used:
- Spring Boot
    - JDK21
    - Maven
- Angular
    - TypeScript
    - HTML
    - CSS
    - Angular 17
- MySQL

[GitHub](https://github.com/jaczerob/eras)

discord-spring-boot-starter is a Spring Boot starter that allows users to easily create Discord bots using Spring Boot.
This starter provides a simple way to create commands using annotations and provides a way to easily interact with the
Discord API.

Technologies used:
- Spring Boot
    - JDK21
    - Maven

[GitHub](https://github.com/jaczerob/discord-spring-boot-starter)
